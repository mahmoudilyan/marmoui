<Accordions>
  <Accordion title="Overview">
    InputShortcode is a specialized text input that supports inserting dynamic shortcodes (merge fields) and optional emoji. It provides a dropdown interface for browsing and inserting shortcodes organized by category.

    **When to Use:**
    - Email subject lines with personalization tokens
    - Message templates with dynamic fields
    - Any text input that supports merge field insertion

    **When NOT to Use:**
    - For standard text input (use Input)
    - For rich text editing (use a Rich Text Editor)
    - When personalization tokens are not needed
  </Accordion>
</Accordions>

## Usage

```tsx
import { InputShortcode } from '@marmoui/ui';
```

## Best Practices

<GuidelinesGrid>
	<Guideline type="do" title="Organize shortcodes by category">
		Group shortcodes logically (Contact, Order, Account) for easy discovery.
	</Guideline>
	<Guideline type="do" title="Show shortcode previews">
		Display a readable label for each shortcode, not just the raw merge field syntax.
	</Guideline>
	<Guideline type="dont" title="Don't overload with options">
		Limit shortcode categories to keep the dropdown manageable.
	</Guideline>
	<Guideline type="dont" title="Don't use without fallback guidance">
		Inform users what happens when a shortcode can't resolve (e.g., empty contact field).
	</Guideline>
</GuidelinesGrid>

## Accessibility

- Shortcode dropdown is navigable via keyboard.
- Inserted shortcodes are part of the input's text value for screen readers.
- Emoji picker (when enabled) supports keyboard navigation.
