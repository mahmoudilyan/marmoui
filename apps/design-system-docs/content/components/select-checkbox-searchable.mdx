<Accordions>
  <Accordion title="Overview">
    SelectCheckboxSearchable combines a dropdown, search input, and checkbox items for selecting multiple options from a potentially large list. Users can search to filter items and toggle checkboxes to make selections.

    **When to Use:**
    - Selecting multiple tags, categories, or filters
    - Assigning multiple users or teams
    - Any multi-select from a list of 10+ items

    **When NOT to Use:**
    - For single selection (use Select)
    - For small option lists (use CheckboxGroup)
    - For free-text multi-value input (use TagsInput)
  </Accordion>
</Accordions>

## Best Practices

<GuidelinesGrid>
	<Guideline type="do" title="Include search for large lists">
		Search is essential when there are more than 7-10 items.
	</Guideline>
	<Guideline type="do" title="Show selected count">
		Display how many items are selected in the trigger when the dropdown is closed.
	</Guideline>
	<Guideline type="dont" title="Don't use for single selection">
		For single-value selection, use the standard Select component.
	</Guideline>
	<Guideline type="dont" title="Don't hide selection state">
		Make it clear which items are checked, even after scrolling.
	</Guideline>
</GuidelinesGrid>

## Accessibility

- Search input is focused when the dropdown opens.
- Checkbox items are navigable with Arrow keys.
- Selected state is announced by screen readers.
- Escape closes the dropdown.
